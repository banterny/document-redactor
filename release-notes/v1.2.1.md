# document-redactor v1.2.1

_Released 2026-04-30_

## What's new

- Legal references are much quieter by default: statutes, contract article numbers, court names, and case citations are no longer treated as redact candidates.
- Case and docket numbers still appear in a dedicated review bucket, but they remain unchecked unless you explicitly select them.
- The in-app footer now reports `v1.2.1`, matching the package version and current release docs.
- The checked single-file HTML build is about **262 KB** and still makes zero network requests.

```bash
sha256sum -c document-redactor.html.sha256
# expected output:

```


## Known limitations

- This is still a deterministic local pre-upload filter, not a legal judgment engine.
- The tool does not decide whether a legal reference is sensitive in your specific matter; review the candidate list before applying redaction.
- DOCX output is verified after rewrite, but unsupported package structures may still need manual handling.
